Start Your Minecraft Server

Follow these simple steps:

1. Log in to GitHub

Go to GitHub and sign in.

2. Open GitHub Codespace

Go to your repository and start a Codespace.

3. Start the Server

In the terminal, type:

./start.sh

4. Connect to Minecraft

Use the IP on the main site to join the server.

Optional: Use Crafty

Go to the Crafty panel and log in.

Optional: Use Tailscale

Download Tailscale, install it, and log in.

Optional: Install Voice Chat

Download and install the voice chat mod for your Minecraft version.
